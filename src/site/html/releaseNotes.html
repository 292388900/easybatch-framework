Release notes

Current stable release

The current CB4J stable release version is 1.0.0. The main features of this release are :

  • Initial core API and Implementation
  • Common data type validators
Download CB4J

Roadmap

Many features are currently developed and will be available in next CB4J versions:

  • version 1.1 :
    1. Add configuration parameters to support data enclosing characters and surrounding white spaces
    2. Add JMX MBeans to support monitoring of batch execution
  • version 1.2 :
    1. Add CB4J-Spring integration module
    2. Add statistics module to report processing time for each record
  • version 2.0 :
    1. Add GUI module that will enable graphical configuration, execution and monitoring of CB4J
    2. Add parallelizer module to split input file into multiple partitions, run parallel CB4J instances and then merge results
