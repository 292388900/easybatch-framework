1. Prerequisite

Easy Batch requires a Java 1.6+ runtime.

2. Build Easy Batch

2.1 Get source code

Easy Batch uses Git as source control manager and is hosted on GitHub. To download Easy Batch source code, clone it's git repository as follows :

git clone https://github.com/benas/easy-batch.git

You can also get the source code in a zip file from the home page of the project on GitHub: https://github.com/benas/easy-batch

2.2 Build with Maven

Easy Batch uses maven as build tool. To build Easy Batch with maven, use the following command from the root directory of the project :

mvn install

3. Use Easy Batch

To use Easy Batch, you have to add easybatch-core-{{ site.version }}.jar to your application's classpath.

If you use maven, add the following dependency to your project pom :

<dependency>
    <groupId>io.github.benas</groupId>
    <artifactId>easybatch-core</artifactId>
    <version>{{ site.version }}</version>
</dependency>
