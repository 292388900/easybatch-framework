Easy Batch tutorials are organized in 3 categories:

  • Basic tutorials: These are a set of Hello World samples which will allow you to understand the basic APIs of Easy Batch and how to use them efficiently.
  • Intermediate Tutorials: These tutorials are real world use cases that show how to extend Easy Batch built-in components on more complex data models and business requirements.
  • Advanced Tutorials: These tutorials show the usage of advanced features of Easy Batch such as processing data in parallel, scheduling Easy Batch instances and Monitoring Easy Batch execution at runtime.
