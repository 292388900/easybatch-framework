1. Introduction

Easy Batch engine implements the java.util.concurrent.Callable interface so it can be easily scheduled using the java.util.concurrent.ScheduledExecutorService API. Nevertheless, this service is limited in terms of scheduling features that's why Easy Batch provides an integration module with the popular Java scheduler Quartz.

In this section, we will introduce how to schedule an Easy Batch instance using the provided EasyBatchScheduler API.

2. Sample usage

First, you need to add the easybatch-quartz module to your project dependencies:

<dependency>
    <groupId>io.github.benas</groupId>
    <artifactId>easybatch-quartz</artifactId>
    <version>{{ site.version }}</version>
</dependency>

Then you can use the EasyBatchScheduler API to schedule Easy Batch executions:

  • At a fixed point of time using scheduleAt(Date when)
  • Repeatedly with predefined interval using scheduleAtWithInterval(Date when, int interval)
  • Using unix cron-like expression with scheduleCron(String cronExpression)

For all details about these APIs, please refer to the Javadoc of the io.github.benas.easybatch.tools.scheduling package.

The listing below shows how to schedule the Hello World tutorial to run every minute:

EasyBatchScheduler easyBatchScheduler = new EasyBatchScheduler(easyBatchEngine);
easyBatchScheduler.scheduleAtWithInterval(new Date(), 1);
easyBatchScheduler.start();

You can find the complete source code of this tutorial here.

To run the tutorial with maven, launch the following command in the easybatch-tutorials directory :

mvn exec:java -PrunQuartzTutorial

If everything is ok, you will see in the console that Easy Batch engine will run every minute.
