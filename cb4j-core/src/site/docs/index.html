CB4J

CSV Batch processing with Java made easy!

Learn more about CB4J

Easy to learn

CB4J has a natural model to deal with CSV files. It also provides a simple API to process CSV files in batch mode. An average Java developer should understand CB4J internals in no more than 2 hours!

Small footprint

CB4J core module consists of only one jar file of 40Ko with no dependencies at all! You have to simply include it in your application's classpath.

Extensible framework

Although it provides default implementation of commons requirements when processing CSV files, CB4j allows you to extend and override almost every default implementation according to your needs.
